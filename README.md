# Trabalhos Computação Gráfica

Os trabalhos estão separados por pastas, para todos a inicialização é feita da mesma forma.

## 🖥️ Como utilizar:

**✨ Primeira incialização**

Acesse o arquivo do trabalho

`cd <nome-do-arquivo>`

Instale os pacotes do npm

`npm i`

**🏃 Rodando a aplicação**

`npm run start`
